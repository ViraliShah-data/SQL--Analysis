# SQL--Analysis
# Online Learning Platform SQL Analysis

## 📌 Project Overview
This project analyzes an online learning platform database using SQL. It focuses on student behavior, course performance, and revenue insights.

## 🗂️ Database Structure
- Students Table
- Courses Table
- Enrollments Table

## 🔍 Key Analysis Performed
- Student demographics analysis
- Course performance and ratings
- Revenue by category
- Most active and valuable students
- Ranking of courses using window functions
- Top spending students by country

## 🛠️ SQL Concepts Used
- JOINs (INNER, LEFT)
- GROUP BY & Aggregations
- Subqueries
- Window Functions (RANK)
- CASE statements

## 📊 Insights
- Data category courses have high engagement
- Some students enroll in multiple courses
- Revenue is concentrated in specific categories
- Top students contribute significantly to revenue

## 🚀 Tools Used
- MySQL / SQL
